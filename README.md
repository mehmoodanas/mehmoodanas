## 🚀 Featured Projects

### 🎓 AI in Customer Service Automation for CRM Systems

> *Bachelor's Thesis · 2024 – 2025*

A transformer-based conversational AI. **DistilBERT** fine-tuned for intent classification and **Sentence-BERT** for semantic similarity retrieval over a curated customer-support corpus. Two-service architecture:

- 🐍 Python / Flask — ML inference micro-service
- ☕ Java / Spring Boot — dialogue backend
- 🌐 HTML/JS — front-end over REST

Comparative evaluation of rule-based vs. DistilBERT vs. Sentence-BERT — benchmarking accuracy, confidence and latency to quantify the interpretability-vs-precision trade-off.

`Python` · `PyTorch` · `Hugging Face` · `DistilBERT` · `Sentence-BERT` · `Flask` · `Java` · `Spring Boot` · `REST`
